Create a Stack of strings to store previous states of text

Function typeText(newText):
    Push current text to stack
    Set current text to newText

Function undo():
    If stack is not empty:
        Pop from stack and set as current text
    Else:
        Print "Nothing to undo"

In main:
    Initialize current text as empty string
    Type some text and push states
    Call undo() to revert to previous text
